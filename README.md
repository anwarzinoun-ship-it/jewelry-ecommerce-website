# Jewelry E-Commerce Website

A responsive jewelry e-commerce website built with HTML, CSS, and JavaScript.

## Features

- Browse jewelry products
- Add products to the shopping cart
- Manage cart items
- Checkout page
- Admin page
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html` - Main page
- `style.css` - Website styling
- `script.js` - Main JavaScript logic
- `products.js` - Product data and management
- `cart.html` / `cart.js` - Shopping cart
- `checkout.html` / `checkout.js` - Checkout system
- `admin.html` / `admin.js` - Admin interface
